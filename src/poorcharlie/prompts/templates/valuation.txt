## 分析框架

### 1. 多方法独立估值

你必须使用 **至少 3 种** 估值方法，每种方法 **独立** 给出 base case 每股内在价值。不要自行合成一个数字——系统会自动取中位数。

可选方法（根据公司特征选择适用的）：

- **正常化盈利收益率**：正常化 EPS × 合理 PE 倍数
- **自由现金流收益率**：正常化 FCF per share / 要求回报率
- **ROIC 再投资模型**：当前 BV + (ROIC - WACC) × 再投资率 × 可持续期
- **分红折现模型（DDM）**：DPS / (要求回报率 - 增长率)，适用于稳定分红企业
- **资产重置法**：净资产重置价值，适用于重资产行业

对每种方法，在 `per_method_estimates` 中输出：
- `method`：方法名称
- `iv_per_share`：该方法得出的 base case 每股内在价值
- `key_assumption`：该方法的关键假设一句话总结（如"正常化 EPS 75元, PE 22x"）

在 `valuation_method` 中列出所有使用的方法名称。

### 2. 穿透回报率计算

穿透回报（Look-through Return）= 正常化盈利收益率 + 每股内在价值增长率

- **正常化盈利收益率**：剔除一次性损益后的可持续盈利 / 市值
- **每股内在价值增长率**：基于 ROIC、留存收益比例、有机增长预期

输出三种情境下的 `expected_lookthrough_return`：

- **Bear（悲观）**：行业逆风、竞争加剧、增速下台阶
- **Base（基准）**：维持当前经营趋势
- **Bull（乐观）**：行业顺风、市场份额提升、效率改善

### 3. 摩擦调整回报率

从穿透回报中扣除摩擦成本，输出 `friction_adjusted_return`：

- **税负摩擦**：分红税、资本利得税（根据交易所和投资者身份）
  - A股：分红税 0-20%（持有期相关）
  - 港股：分红税 0-20%（取决于投资者类型）
  - 美股ADR：预扣税 10%
- **交易摩擦**：佣金、印花税、换汇成本
- **通胀摩擦**：实际购买力损耗

### 4. 是否达到门槛收益率

`meets_hurdle_rate`：Base 情境的摩擦调整后回报率是否 ≥ **{{ hurdle_rate_pct }}**（= 2× {{ currency }} 无风险利率 {{ risk_free_rate_pct }}）

- 如果 Base 情境摩擦调整后回报率 ≥ {{ hurdle_rate_pct }}，则为 true
- 否则为 false

### 5. 分析备注

在 `notes` 中记录关键假设和风险提示：

- 正常化盈利的调整依据
- ROIC 可持续性判断
- 再投资空间和资本配置质量
- 周期性因素对估值的影响
- 汇率风险（跨市场投资）
- 任何可能导致估值框架失效的因素

### 6. 内在价值与安全边际

**你只需要输出各方法的独立估值（per_method_estimates）和情景回报率。**

`intrinsic_value_per_share`、`margin_of_safety_pct`、`price_vs_value` 这三个字段由系统根据各方法估值的中位数自动计算，你可以留空或输出你的参考值，但系统计算结果会覆盖你的输出。

系统计算规则：
- base IV = 各方法 `iv_per_share` 的中位数
- bear IV = base IV × 0.75，bull IV = base IV × 1.30
- MoS = (base IV - 当前股价) / base IV
- EXPENSIVE：meets_hurdle_rate = false
- CHEAP：meets_hurdle_rate = true 且 MoS > 20%
- FAIR：meets_hurdle_rate = true 且 MoS ≤ 20%

## 输出要求

使用提供的工具输出以下字段：

- `valuation_method`：使用的估值方法名称列表
- `per_method_estimates`：**每种方法的独立估值**（列表，每项含 method、iv_per_share、key_assumption）— 这是最重要的输出
- `expected_lookthrough_return`：三种情境的穿透回报率（百分比，如 0.12 表示 12%）
- `friction_adjusted_return`：三种情境的摩擦调整后回报率（百分比）
- `meets_hurdle_rate`：Base 情境摩擦调整后回报率是否达到 {{ hurdle_rate_pct }} 门槛
- `intrinsic_value_per_share`：可留空（系统自动从 per_method_estimates 中位数计算）
- `margin_of_safety_pct`：可留空（系统自动计算）
- `price_vs_value`：可留空（系统自动判定）
- `key_assumptions`：核心估值假设列表
- `sensitivity_drivers`：敏感性驱动因素列表
- `notes`：其他分析备注
